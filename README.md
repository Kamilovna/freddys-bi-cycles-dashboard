# Freddys BI-Cycles — Power BI Dashboard
## Overview
This repository contains a Power BI dashboard developed for sales analysis within the fictional company Freddys BI-Cycles. The dashboard provides a comprehensive view of sales performance across regions, product categories, and time periods. It is designed to support data-driven decision-making by visualizing key metrics such as revenue, costs, margins, and order volumes.
### Features
- **Sales Overview**: Total sales, costs, margin, order count, and average daily sales
- **Time Series Analysis**: Year-over-year sales trends
- **Geographical Insights**: Sales by continent, country, and interactive world map
- **Category Breakdown**: Bikes, Components, Clothing, Accessories
- **Top Product Colors**: Ranking of best-selling colors
- **Dynamic Filters**: Calendar year selection for flexible analysis
